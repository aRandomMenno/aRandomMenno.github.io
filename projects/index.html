Goober OS

Goober OS is a web based Terminal. With a command to open a windows 95/98 Inspired GUI. I made this in collaboration with Jea (my sister), Stone, Tigo and Rianne. It was a school project for a Make-a-thon. We had 48 hours to make a project in the theme of Lo-Fi Retro. And always 1 person had to be programming. We had a lot of fun making this project. I mostly worked on the logic for the commands in the terminal.
