Improve Comment finder extension

Some time ago I made an improvement to an existing extension called HTML Comment Finder (for firefox). Which my sister forked from the Chrome version. I made some improvements to the extension improving readability and accidentally introduced a security vulnerability.

The security vulnerability in the extension would make it so that any commented out script tag would be executed when the extension was enabled. Like this would be executed:
<script src="https://example.com/malicious_script.js"><script>
This was promptly fixed by my sister.
