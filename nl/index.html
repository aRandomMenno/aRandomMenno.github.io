Welkom op mijn website. Ik ben 18 jaar en studeer Software Development op het Grafisch Lyceum Rotterdam deze opleiding duurt 3 jaar, momenteel zit ik in het tweede jaar.

Programmeren & Tech

Op dit moment ben ik redelijk vaardig in drie programmeertalen. De taal waar ik me het meest thuis in voel is PHP, daarna JavaScript en vervolgens Python. Mijn Python-kennis kan wel een opfrisbeurt gebruiken, omdat ik het sinds mijn “simpele” project om een deel van een mobiel spel te automatiseren nauwelijks heb gebruikt. Voor school onderzoek ik nu vier talen: Kotlin vs Java voor een Minecraft-mod, of Rust vs Go om een schaakcomputer te bouwen. Binnenkort moet ik kiezen; op dit moment neig ik naar Rust.

Ik werk nu zo'n tweeënhalf jaar met JavaScript en voel me er vrij zeker in. Ik pas eenvoudig de DOM aan, haal data op met Fetch en schrijf asynchrone scripts. Ook ken ik twee libraries/frameworks goed: P5.js en GSAP. In PHP heb ik ervaring met PHPMyAdmin en MariaDB. Ik maakte meerdere kleine projecten met PHP en databases; we begonnen met Sqlite en stapten later over op MariaDB.

Git, GitHub en Linux gebruik ik dagelijks. Met Git hou ik al mijn (school)projecten bij. Op GitHub staan enkele repositories; een paar zijn privé. Linux gebruik ik al zo'n anderhalf jaar als mijn primaire OS. Op mijn Raspberry Pi draait OwnTracks en in de toekomst wil ik een Nextcloud-instantie hosten.

In mijn vrije tijd luister ik graag naar muziek, game ik, slaap ik, en maak ik foto's. Ik heb een fotografiecursus gevolgd en vind fotograferen erg leuk. Een selectie staat in mijn galerij.

De toekomst

Na mijn huidige studie wil ik graag Cyber Security studeren, met focus op ethical hacking, netwerkverdediging of het bestrijden van social engineering. Mocht de techwereld niets voor mij zijn, dan kan ik altijd de fotografie in, want ook dat doe ik met plezier.
